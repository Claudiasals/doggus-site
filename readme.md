# Doggus Website

Static website developed for **Doggus**, a handmade brand focused on personalized pet tags.

The project is intentionally designed as a **vetrina artigianale (showcase website)** rather than a traditional e-commerce, allowing users to explore styles and contact directly for custom creations.

---

## ✨ Concept

This website presents a curated collection of styles, materials and inspirations.

Each product is:
- handmade
- created on request
- developed through direct communication

There is **no automatic checkout or purchase system**, in order to keep the experience personal and flexible.

---

## 🎯 Features

- Responsive design (mobile-first)
- Clean and modern UI with soft gradients and spacing
- Lightweight animations for better UX
- Style-based navigation (themes instead of catalog)
- Contact form for custom requests (Netlify Forms ready)
- Legal pages structure (privacy, terms, cookies)

---

## 🧱 Tech Stack

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Netlify (deployment & form handling)
- GitHub (version control)

---

## 📁 Project Structure

- `/index.html` → Homepage
- `/themes/` → Style collections (Montagna, Floreale, Mare, etc.)
- `/assets/` → Images and logo
- `/style.css` → Main styling
- `/script.js` → Animations and interactions
- `/privacy.html`, `/termini.html`, etc. → Legal pages

---

## 🧠 Design Approach

The layout focuses on:
- **visual breathing space** (no crowded UI)
- soft color palettes inspired by the brand
- modular sections easy to expand
- a balance between aesthetic and clarity

---

## 📩 Contact Flow

Users can:
1. explore styles
2. get inspired
3. send a message via form

All requests are handled manually via:
- email
- WhatsApp (optional)
