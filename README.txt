README
------
Questa cartella sostituisce integralmente la versione precedente del sito.

Come aggiornare:
1. apri questa cartella
2. copia tutto dentro la tua cartella progetto locale
3. sostituisci i file esistenti
4. esegui:

git add .
git commit -m "premium redesign hobby friendly"
git push

Netlify aggiornerà il sito automaticamente.
